# Robinhood L2 — Token Explorer

Локальное веб-приложение: вводишь адрес токена на **Robinhood L2** (chainId `4663`)
и видишь **маркеткап, покупки и продажи** этого токена с встроенного DEX.

Работает напрямую через RPC цепи (Arbitrum Nitro / Orbit) + `ethers` v6 — без
сторонних API и ключей. DEX на цепи — Uniswap V2-совместимый, данные о сделках
читаются из событий `Swap`, цена — из резервов пула.

## Что показывает

- Имя, символ, адрес, total supply токена.
- **Market cap** = цена × total supply (цена берётся из пула против WETH).
- **Цена** токена в ETH (или в USD, если задан `ETH_USD`).
- Ликвидность пула (сторона ETH).
- Счётчики и объёмы **покупок / продаж**.
- Таблицу последних сделок: тип (BUY/SELL), объём, цена, время, трейдер, ссылка на tx
  в эксплорере.

## Требования

- **Node.js 18+** (`node -v`).

## Запуск (Windows / PowerShell)

```powershell
cd Z:\rh-token-explorer
npm install
copy .env.example .env
npm start
```

Открой в браузере: **http://localhost:3000**

Вставь адрес токена (например, один из торгующихся на цепи) и нажми «Показать».

## Конфигурация (`.env`)

| Переменная | Назначение |
|---|---|
| `RPC_URL` | RPC Robinhood L2. По умолчанию `https://poptye-always-win.poptyedev.com/`. |
| `CHAIN_ID` | `4663`. |
| `EXPLORER_URL` | Базовый URL эксплорера для ссылок на tx/адреса. |
| `WETH_ADDRESS` | Котировочный токен (WETH), обнаружен на цепи: `0x0bd7d308…cad73`. |
| `ETH_USD` | Необязательно. Если задать цену ETH в USD — маркеткап и объёмы будут и в долларах. Пусто = всё в ETH. |
| `PORT` | Порт локального сервера (по умолчанию `3000`). |

## Как это работает

1. По адресу токена читаются ERC-20 поля (`name/symbol/decimals/totalSupply`).
2. Через события `PairCreated` (по индексированным топикам) находится DEX-пул токена;
   предпочтение отдаётся паре против WETH.
3. Из `getReserves()` пула считается цена (quote за 1 токен) и маркеткап.
4. Из событий `Swap` этого пула декодируются сделки и классифицируются на
   покупки (токен уходит из пула трейдеру) и продажи (токен приходит в пул).

## Ограничения / честные оговорки

- **Market cap в ETH, не в USD**, пока не задашь `ETH_USD`. Я не подключал внешний
  прайс-фид, чтобы приложение оставалось самодостаточным и без внешних зависимостей.
- Если токен спарен не с WETH, маркеткап считается в его котировочном токене (об этом
  будет предупреждение в интерфейсе).
- Учитываются только сделки через DEX-пулы (события `Swap`). Обычные переводы
  (`Transfer`) не считаются покупкой/продажей.
- Если у токена нет пула — покажутся только метаданные, маркеткап/сделки будут пустыми.
- Встроенный эксплорер цепи агрессивно режет запросы (HTTP 429), поэтому приложение
  специально на него **не опирается** — все данные берутся из RPC. Эксплорер
  используется только для гиперссылок.
